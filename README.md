# REFA AI - Legal Pages

Этот репозиторий содержит юридические страницы для приложения REFA AI:
- Terms of Service
- Privacy Policy
- Refund Policy
- Pricing Page

## GitHub Pages

Сайт автоматически публикуется на: https://refa-ai.github.io

## Страницы

- Terms of Service: https://refa-ai.github.io/terms.html
- Privacy Policy: https://refa-ai.github.io/privacy.html
- Refund Policy: https://refa-ai.github.io/refund.html
- Pricing: https://refa-ai.github.io/pricing.html

## Настройка GitHub Pages

1. Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: `main`, Folder: `/ (root)`
4. Save
